使用Markdown书写文章

2017-09-01
#Technology
2017-9

Markdown 是一种被设计成为适合于网络书写的”语言”,您现在看到的这篇文章就是使用Markdown书写的。
2004年由John Gruber和Aaron Swartz设计,其目的在于为普通写作者提供更为方便的书写方式。
Markdown的语法非常简单。2016年发布的RFC 7763正式将其作为一种新的MIME类型 text/markdown。

Markdown其实只是支持一小部分HTML,比如在Markdown中,我们想强调内容,可以这样写:

i’m strong! * i’m strong! *

也可以写HTML标签

i’m strong! < strong >i’m strong!< /strong >

不过既然我们使用Markdown就不应该再使用标签风格！

标题

Markdown 支持两种形式的标题语法 类Setext和类atx形式
类Setext

任意数量=或者任意数量-写在标题文字之下

类atx

1-6个#之后写标题,#xx最大,######xx最小

区块引用

here is a block quote!
here too!

embeded block

无序列表

使用 *,-,+都可以(后跟空格)

  • one
  • two
  • third
  • four
  • five
    可以嵌套 > 使用,需要缩进
  • i’here
    i’here

    i’here

有序列表

数字加英文点.(后跟空格)

  1. hi
  2. xi

分隔线

三个或以上的,-,_ 和-之间可以加空格

[a hidden tab]here is a block!  
-------  
i'm ugly!  
* * *  
but kind!  
_____  
so, do you love me?

i’m ugly!

but kind!

so, do you love me?

换行

句末加两个连续的空格！

链接

Markdown 支持行内式和参考式
[linkname](URL “title”) title的作用就是鼠标放上去的提示,可选
[linkname][id]
[define a id]:[空格] URL title

  1. a link to google:) link
  2. a link to my blog:) click

强调

*a a
*a a
_b_ b
__b b__

图片

同样支持行内式和参考式

![linkname](URL “title”) title的作用就是鼠标放上去的提示,可选
![linkname][id]
[define a id]:[空格] URL title

目前Markdown语法上不支持设定图片宽高,只能使用 \

自动链接

可以这样写
< guxiayu@protonmail.com >:
guxiayu@protonmail.com

反斜线

比如我想在文章中写 **please don’t parse me!
我可以这样写 \*\*please don’t parse me!

tab代码区块

[a hidden tab]here is a block!
比`更方便:)
<html></html>
**ewfe**
[]()

代码

` content `

# include <stdio.h>
如果要在中间插入反引号,可以使用多个反引号
print "i love timo'mushroom",

表格

其实用Markdown写表格确实不方便,还不如用HTML呢:)
用冒号选择对齐方式 不写默认居左
:—: 居中
—-: 居右

\\|name    |age    |gender |
  |--------|:-----:|------:|
  |maple   |20     |man    |
name age gender
maple 20 man

你可以看出是不是不方便啊:),而且只能用在简单场合
所以用HTML吧
